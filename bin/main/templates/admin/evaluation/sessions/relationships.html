평가 관계 관리

RULE_BASED 모드입니다. 관계 재생성은 이 화면의 자동 생성에서 수행됩니다. 세션 시작(start)은 이미 생성된 snapshot으로 최종 관계/assignment를 확정합니다.
관계 자동 생성 화면 안내
  • 이 화면에서 하는 일: 세션 관계 자동 생성 실행, 최종 관계 검토, 수동 ADD/REMOVE override 적용.
  • 입력해야 하는 값: 수동 추가 시 평가자/피평가자, 필터 조회 조건.
  • 자동 처리되는 값: RULE_BASED 실행 시 EXCLUDE/자기평가 제거/중복 제거/override 병합 후 최종 관계 반영.
  • auto-generate와 start 역할: 룰/데이터/override 변경사항은 자동 생성을 다시 실행해야 반영됩니다. start는 자동 재계산을 수행하지 않습니다.
  • 자주 발생하는 실수: override 변경 후 재생성 누락, source_rule 근거 확인 누락, 실행 이력 확인 없이 세션 시작.
  • 저장/실행 전 확인: 실행 이력 건수(생성/제외/중복제거/override), 최종 관계 샘플, LEGACY 비교 결과를 확인하세요.
  • 문항군 유의사항: 실제 평가 화면에는 assignment에 스냅샷 저장된 `resolved_question_group_code` 기준 문항만 노출되며, start 이후에는 고정됩니다.
