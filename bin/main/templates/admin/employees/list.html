직원 관리/업로드 운영 안내
  • 이 화면에서 하는 일: 직원 수기 등록, 직원 엑셀 업로드(표준 템플릿/운영 파일 자동 인식), 업로드 전 미리보기 검증.
  • 입력해야 하는 값: 사원번호, 이름, 부서코드 또는 부서명 중 최소 1개, 필요 시 상태/속성 컬럼.
  • 자동 처리되는 값: `login_id=사번`, 상태 미입력 시 `ACTIVE`, boolean 값은 `Y/N`, `TRUE/FALSE`, `1/0` 정규화.
  • 자주 발생하는 실수: 부서코드/부서명 불일치, 동일 이름 부서 중복(모호), 부서명 오탈자, boolean 오입력.
  • 저장/실행 전 확인: 미리보기에서 파일 유형/적용 프로파일/누락 필수 컬럼/부서 매칭 결과를 먼저 확인한 뒤 업로드하세요.
  • 부서 매칭 정책: 코드 우선, 코드 없으면 이름 매칭, 코드+이름 동시 입력 시 동일 부서 검증(불일치 시 오류).
