# 관리자 매뉴얼 초안

## 1. 로그인 및 권한

- 최고 관리자: 기관/기관 관리자 운영
- 기관 관리자: 조직/평가 운영
- 일반 사용자: 평가 작성/제출

권한 오류 발생 시:
- URL 직접 접근이 막히는지 확인
- 요청 ID를 운영팀에 전달

## 2. 기관 관리자 업무 순서

1. 부서 등록(수동/엑셀)
2. 직원 등록(수동/엑셀)
3. 평가 템플릿/문항 구성
4. 평가 세션 생성
5. 평가 관계 자동 생성 후 수동 보정
6. 세션 시작 및 진행 모니터링
7. 세션 종료

## 3. 운영 중 자주 보는 화면

- `/admin/uploads/history`: 업로드 성공/실패 추적
- `/admin/evaluation/sessions`: 세션 상태 확인
- `/admin/evaluation/sessions/{id}`: 미제출자/진행률 확인
- `/admin/evaluation/sessions/{id}/relationships`: 관계 보정

## 4. 오류 대응

1. 화면 메시지 확인
2. 요청 ID 확인
3. 감사로그/애플리케이션 로그 조회
4. 동일 작업 재시도 전 데이터 중복 여부 확인

## 5. 운영 주의사항

- 세션 시작 후 관계 수정은 제한됨
- 재제출 허용 여부는 세션 정책을 따름
- 운영 환경에서는 초기 관리자 자동 생성 비활성화 유지
