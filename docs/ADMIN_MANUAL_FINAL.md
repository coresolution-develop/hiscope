# 관리자 매뉴얼 (최종본)

기준일: 2026-03-11

## 1. 역할

- 최고 관리자(`ROLE_SUPER_ADMIN`)
  - 기관 생성/상태 변경
  - 기관 관리자 계정 발급
  - 전기관 감사로그 조회
- 기관 관리자(`ROLE_ORG_ADMIN`)
  - 부서/직원/평가 템플릿/세션/관계/업로드 운영

## 2. 표준 운영 순서

1. 부서 등록 (수기/엑셀)
2. 직원 등록 (수기/엑셀)
3. 템플릿/문항 구성
4. 세션 생성
5. 평가관계 자동생성 후 수동 보정
6. 세션 시작
7. 진행 모니터링
8. 세션 종료

## 3. 일일 점검

- `/admin/uploads/history`: 업로드 실패/부분성공 여부
- `/admin/evaluation/sessions`: 세션 상태
- `/admin/evaluation/sessions/{id}`: 미제출자/진행률
- `/admin/audit`: 주요 이벤트 및 비정상 시도 확인

## 4. 운영 규칙

- 세션 시작 후 관계 변경 제한
- 기관 간 데이터 접근 금지 (URL 직접 접근 포함)
- 업로드 실패 시 이력 확인 후 재업로드
- 오류 대응은 요청 ID 기반 추적

## 5. 보안 규칙

- 관리자 계정 공유 금지
- 비밀번호 주기 변경
- 운영에서는 초기 슈퍼관리자 자동 생성 비활성화 유지
