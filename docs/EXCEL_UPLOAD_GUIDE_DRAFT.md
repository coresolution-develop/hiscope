# 엑셀 업로드 가이드 초안

## 공통 규칙

- 파일 형식: `.xlsx`만 허용
- 파일 크기: 기본 10MB 이하
- 헤더는 템플릿 형식을 유지
- 대량 업로드 전 샘플 5~10건으로 사전 검증 권장

## 1. 부서 업로드

### 컬럼
- 부서코드
- 부서명
- 상위부서코드(선택)

### 실패 사례
- 부서코드 형식 오류
- 같은 파일 내 중복 부서코드
- 존재하지 않는 상위부서코드

## 2. 직원 업로드

### 컬럼
- 사원번호
- 이름
- 부서코드
- 직위
- 직책
- 이메일
- 로그인ID
- 상태(ACTIVE/INACTIVE/LEAVE)

### 실패 사례
- 존재하지 않는 부서코드
- 중복 로그인ID / 사원번호
- 상태 허용값 외 입력

## 3. 평가 문항 업로드

### 컬럼
- 카테고리
- 문항내용
- 문항유형(SCALE/DESCRIPTIVE)
- 최대점수(SCALE 필수)
- 정렬순서

### 실패 사례
- 문항유형 허용값 오류
- SCALE인데 최대점수 누락
- 숫자 필드 형식 오류

## 4. 업로드 실패 시 처리

1. 업로드 결과 메시지 확인
2. 업로드 이력 화면에서 실패 상세 확인
3. 원본 파일 수정 후 재업로드
4. 반복 실패 시 요청 ID와 함께 운영팀 문의
