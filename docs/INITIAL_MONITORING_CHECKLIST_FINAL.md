# Initial Monitoring Checklist (Day 1-3)

기준일: 2026-03-11

## Day 1 (배포 당일)

### 서비스 상태
- [ ] 15분 간격 헬스체크 확인
- [ ] 프로세스/컨테이너 재시작 여부 확인

### 에러/로그
- [ ] 5xx 발생 추이 확인
- [ ] 주요 예외(권한, 업로드, 제출) 빈도 확인
- [ ] requestId 기반 미해결 오류 0건 목표

### 기능
- [ ] 로그인 실패율 급증 여부 확인
- [ ] 업로드 실패율 확인(정상 범위 이탈 여부)
- [ ] 평가 제출 성공률 확인

## Day 2

### 데이터/보안
- [ ] 기관별 데이터 혼입 징후 점검
- [ ] 권한 거부(403) 패턴 점검(비정상 접근 급증 여부)
- [ ] 감사로그 적재 누락 여부 점검

### 성능
- [ ] 느린 화면(직원목록/세션/감사로그) 응답시간 샘플링
- [ ] DB 슬로우 쿼리 여부 점검

## Day 3

### 안정화 판단
- [ ] 장애 티켓 추이 감소 확인
- [ ] 반복 이슈 RCA/재발방지 액션 등록
- [ ] 운영 보고서 작성(이슈/조치/잔여 리스크)

## Daily KPI(권장)

- 로그인 성공률
- 업로드 성공률
- 평가 제출 성공률
- 5xx 건수
- 평균/95p 응답시간
- 감사로그 적재 건수
