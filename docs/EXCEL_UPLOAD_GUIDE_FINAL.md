# 엑셀 업로드 가이드 (최종본)

기준일: 2026-03-11

## 1. 공통 정책

- 파일 형식: `.xlsx`
- 파일 크기: `APP_UPLOAD_MAX_FILE_SIZE_BYTES` 이하 (기본 10MB)
- 최대 행수: `APP_UPLOAD_MAX_ROWS` 이하 (기본 5,000행)
- 결과 상태: `SUCCESS` / `PARTIAL` / `FAILED`

## 2. 부서 업로드

컬럼:
1. 부서코드(필수)
2. 부서명(필수)
3. 상위부서코드(선택)

주요 검증:
- 코드 형식 검증
- 파일 내 중복 코드 금지
- 상위부서 참조 유효성

## 3. 직원 업로드

컬럼:
1. 사원번호
2. 이름
3. 부서코드
4. 직위
5. 직책
6. 이메일
7. 로그인ID
8. 상태(ACTIVE/INACTIVE/LEAVE)

주요 검증:
- 부서코드 존재 여부
- 로그인ID 형식/중복
- 사원번호 중복

## 4. 문항 업로드

컬럼:
1. 카테고리
2. 문항내용
3. 문항유형(SCALE/DESCRIPTIVE)
4. 최대점수(SCALE 필수)
5. 정렬순서

주요 검증:
- 문항유형 허용값
- 숫자형식 검증

## 5. 실패 처리 절차

1. 화면 에러 확인
2. `/admin/uploads/history` 실패 상세 확인
3. 파일 수정 후 재업로드
4. 반복 실패 시 요청 ID와 함께 운영팀 전달
